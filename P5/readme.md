# Proyecto P5

Este proyecto consiste en una página sencilla en la que se pueden cargar imágenes aleatorias al hacer clic en un botón. La interfaz muestra un contenedor donde se agregan nuevas imágenes dinámicamente mediante JavaScript.

## Funcionalidad
- Al presionar el botón, se generan varias imágenes aleatorias.
- Cada imagen se muestra en la página sin recargarla.
- El proyecto demuestra el uso de eventos, creación de elementos HTML y manipulación del DOM.

## Objetivo didáctico
Este ejercicio está pensado para programadores principiantes que desean practicar cómo interactuar con el DOM en JavaScript. A través de este proyecto, se aprende a:
- escuchar eventos como clics en botones;
- crear elementos HTML dinámicamente;
- agregar contenido a la página mediante programación;
- comprender cómo funciona la relación entre HTML, CSS y JavaScript.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
