# Sendero del corazón

Este proyecto es una pequeña experiencia visual interactiva creada con HTML, CSS y JavaScript. Al mover el cursor sobre la página, aparecen elementos dinámicos que se generan en la posición del mouse, creando un efecto llamativo y romántico.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidad
- Efecto interactivo al pasar el mouse
- Elementos generados dinámicamente en pantalla
- Diseño simple y visualmente atractivo
