# Proyecto P2
Este proyecto muestra una animación de texto simple en la página web. El contenido cambia de forma dinámica usando JavaScript para crear un efecto visual básico.

## Contenido
- Archivo HTML: index.html
- Archivo CSS: styles.css
- Archivo JavaScript: main.js

## Objetivo
Practicar la manipulación del DOM y la creación de animaciones con JavaScript.
