# Prácticas de Full Stack para principiantes

Bienvenid@ a este repositorio, donde comienzo mi camino en el desarrollo web con ejercicios prácticos y sencillos. Aquí voy consolidando mis primeros conocimientos en HTML, CSS, JavaScript y conceptos básicos de Full Stack.

## ¿Qué es este proyecto?

Este espacio reúne una serie de prácticas diseñadas para aprender paso a paso el mundo del desarrollo web. Cada carpeta representa una actividad que me ayuda a entender mejor cómo construir interfaces, manejar eventos, trabajar con JavaScript y organizar proyectos de forma clara.

## Objetivo

El propósito de estas prácticas es convertir los conceptos teóricos en experiencia real. Aprender haciendo, probando ideas y mejorando poco a poco hasta crear proyectos más completos.

## Lo que estoy aprendiendo

- Fundamentos de HTML y CSS
- Lógica básica con JavaScript
- Manipulación del DOM
- Trabajo con formularios y eventos
- Estructura y organización de proyectos web
- Introducción al pensamiento de desarrollo Full Stack

## Prácticas incluidas

- P1: Introducción a la estructura básica de una página web
- P2: Estilos y diseño inicial
- P3: Interactividad con JavaScript
- P4: Manejo de eventos y acciones del usuario
- P5: Prácticas de lógica y funciones
- P6: Trabajo con elementos dinámicos


## Mensaje personal

Este repositorio no solo representa ejercicios, sino también mi progreso como principiante en Full Stack. Cada práctica me enseña a pensar de forma más creativa, lógica y organizada como desarrollador.

¡Estoy empezando, pero con ganas de seguir aprendiendo y mejorando cada día!
