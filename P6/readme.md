# Proyecto P6

Este proyecto es una pequeña práctica de JavaScript para crear un interruptor de modo oscuro. Al hacer clic en el checkbox, la página cambia de fondo y color de texto para simular un tema oscuro.

## Funcionalidad
- Permite alternar entre modo claro y modo oscuro.
- Cambia visualmente la apariencia de la página al activar el interruptor.
- Guarda la preferencia del usuario mediante localStorage para que se recuerde al recargar la página.

## Objetivo didáctico
Este ejercicio está pensado para principiantes que quieren practicar:
- el uso de eventos en JavaScript;
- la manipulación del DOM;
- el manejo del estado de un checkbox;
- el almacenamiento de preferencias con localStorage.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
