# Mini Calendario

Este proyecto muestra un pequeño calendario que presenta el mes, el día de la semana, el número del día y el año actual en español. Usa HTML, CSS y JavaScript para obtener la fecha del sistema y actualizar el contenido en la página.

- `index.html`: estructura del calendario.
- `styles.css`: estilos visuales.
- `index.js`: lógica para obtener la fecha actual y mostrarla en pantalla.

Es ideal para aprender a manipular fechas en JavaScript y trabajar con elementos del DOM.
