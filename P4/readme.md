# Botón Animado

Este proyecto es un botón con un efecto que aparece donde toca el cursor.

- `index.html`: contiene el botón.
- `styles.css`: da el diseño y la animación.
- `index.js`: calcula dónde está el mouse dentro del botón.

El botón usa el evento `mouseover` para saber la posición del cursor y mover la animación al punto de contacto.

Es un ejemplo simple de cómo usar JavaScript y CSS juntos para crear un efecto interactivo.
